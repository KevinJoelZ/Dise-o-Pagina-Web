¿Qué es Flexbox y cómo se utiliza?

Flexbox es un modelo de diseño que proporciona una forma más eficiente de distribuir el espacio entre los elementos de un contenedor, incluso cuando su tamaño es desconocido o dinámico.


¿Por qué flexbox? Durante mucho tiempo, las únicas herramientas fiables con compatibilidad cruzada entre navegadores disponibles para crear diseños CSS fueron cosas como la flotación y el posicionamiento. Están bien y funcionan, pero de alguna manera también limitan bastante y frustran. Con tales herramientas resulta difícil, si no imposible, lograr obtener en cualquier forma conveniente y flexible un diseño de página sencillo con unos requisitos como los siguientes:

Centrar verticalmente un bloque de contenido dentro de su elemento padre. Hacer que todos los elementos secundarios de un contenedor ocupen una cantidad igual del ancho/alto disponible, independientemente del ancho/alto que haya disponible. Hacer que todas las columnas en una compaginación en columnas múltiples adopten la misma altura incluso si contienen cantidades diferentes de contenido. Como verás en las secciones siguientes, los elementos flexbox facilitan mucho algunas tareas de compaginación. ¡Vamos a profundizar en ello!

Características de Flexbox


Flexbox (Flexible Box Layout) es un modelo de diseño de CSS3 que permite crear layouts más eficientes y predecibles al distribuir el espacio entre los elementos de un contenedor. Aquí están las principales características y propiedades de Flexbox:

Contenedor Flex


Para activar Flexbox en un contenedor, se utiliza la propiedad display: flex; o display: inline-flex;.

Eje Principal y Eje Transversal Eje Principal (Main Axis): Por defecto es horizontal (de izquierda a derecha). Es la dirección en la que los elementos flex se colocan.

Eje Transversal (Cross Axis): Es perpendicular al eje principal (por defecto, vertical). Propiedades del Contenedor Flex

flex-direction Define la dirección en la que los elementos flex se colocan dentro del contenedor.

row (por defecto): De izquierda a derecha. row-reverse: De derecha a izquierda. column: De arriba a abajo. column-reverse: De abajo a arriba.

flex-wrap Controla si los elementos flexibles deben colocarse en una sola línea o pueden dividirse en varias líneas.

nowrap (por defecto): Todos los elementos en una línea. wrap: Los elementos se envuelven en varias líneas. wrap-reverse: Los elementos se envuelven en varias líneas, pero en orden inverso.

flex-flow Una propiedad abreviada para flex-direction y flex-wrap.

Ejemplo: flex-flow: row wrap;

justify-content Define la alineación de los elementos a lo largo del eje principal.

flex-start: Los elementos se alinean al inicio del contenedor. flex-end: Los elementos se alinean al final del contenedor. center: Los elementos se centran en el contenedor. space-between: Se deja espacio igual entre los elementos. space-around: Se deja espacio igual alrededor de los elementos. space-evenly: Se deja espacio igual entre los elementos, incluyendo el inicio y el final del contenedor.

align-items Define la alineación de los elementos a lo largo del eje transversal.

stretch (por defecto): Los elementos se estiran para llenar el contenedor. flex-start: Los elementos se alinean al inicio del eje transversal. flex-end: Los elementos se alinean al final del eje transversal. center: Los elementos se centran en el eje transversal. baseline: Los elementos se alinean según su línea base.

align-content Define la alineación de las líneas flexibles cuando hay espacio adicional en el contenedor.

flex-start: Las líneas se alinean al inicio del contenedor. flex-end: Las líneas se alinean al final del contenedor. center: Las líneas se centran en el contenedor. space-between: Se deja espacio igual entre las líneas. space-around: Se deja espacio igual alrededor de las líneas. stretch (por defecto): Las líneas se estiran para llenar el contenedor.

Propiedades de los Elementos Flex



order Define el orden de los elementos flexibles. El valor por defecto es 0, pero se puede cambiar para reordenar los elementos.

Ejemplo: order: 2;

flex-grow Define la capacidad de un elemento para crecer si es necesario. El valor por defecto es 0.

Ejemplo: flex-grow: 1;

flex-shrink Define la capacidad de un elemento para encogerse si es necesario. El valor por defecto es 1.

Ejemplo: flex-shrink: 2;

flex-basis Define el tamaño base principal de un elemento antes de que se distribuya el espacio disponible. El valor por defecto es auto.

Ejemplo: flex-basis: 100px;

flex Una propiedad abreviada para flex-grow, flex-shrink y flex-basis.

Ejemplo: flex: 1 1 100px;

align-self Permite anular la propiedad align-items para elementos específicos.

Ejemplo: align-self: flex-end;

  • display: Define un contenedor flexible.
  • flex-direction: Establece la dirección del eje principal.
  • justify-content: Define la alineación de los elementos a lo largo del eje principal.
  • align-items: Define la alineación de los elementos a lo largo del eje transversal.
  • flex-wrap: Controla si los elementos flexibles deben colocarse en una sola línea o pueden dividirse en múltiples líneas.

¿Qué es Grid y cómo se utiliza?

CSS Grid Layout es un sistema bidimensional para crear diseños de cuadrícula en CSS. Es ideal para crear layouts complejos con facilidad.

Uno de los procesos más problematicos y frustrantes de CSS, sobre todo para novatos o principiantes, es el proceso de colocar y distribuir los elementos a lo largo de una página. Mecanismos como posicionamiento, floats o elementos en bloque o en línea, suelen ser insuficientes (o muy complejos) para crear un layout o estructuras para páginas web actuales.

El sistema de elementos flexibles Flex es una gran mejora, sin embargo, está orientado a estructuras de una sola dimensión, y puede ser laborioso crear estructuras más complejas, por lo que aún necesitamos algo más potente para estructuras web de varias dimensiones rápidamente. Con el paso del tiempo, muchos frameworks CSS y librerías comenzaron a utilizar un sistema basado en un grid donde se definía una cuadrícula, a la que era posible darle tamaño, posición o colocación, cambiando el nombre de sus clases.

Grid CSS nace de esa necesidad, obteniendo las ventajas de ese sistema grid, añadiéndole numerosas mejoras y características que permiten crear rápidamente cuadrículas flexibles y potentes de forma prácticamente instantánea con una nueva familia de propiedades CSS.

Características de CSS Grid

CSS Grid Layout es un sistema de diseño bidimensional que permite crear layouts complejos en la web. Aquí están las principales propiedades y características:

Contenedor Grid Para activar el grid en un contenedor, se utiliza la propiedad display: grid;.

Propiedades del Contenedor Grid

grid-template-columns y grid-template-rows Define el número y tamaño de las columnas y filas.

Ejemplo: grid-template-columns: 100px 200px; Ejemplo: grid-template-rows: auto 50px;

grid-gap Define el espacio entre filas y columnas. Ejemplo: grid-gap: 10px;

grid-template-areas Permite nombrar áreas dentro de la cuadrícula

justify-items Alinea los elementos hijos a lo largo del eje horizontal.

Valores: start, end, center, stretch.

align-items Alinea los elementos hijos a lo largo del eje vertical.

Valores: start, end, center, stretch. justify-content y align-content Alinean el contenido de la cuadrícula dentro del contenedor.

Ejemplo: justify-content: space-between; Ejemplo: align-content: center;

Propiedades de los Elementos Grid

grid-column y grid-row Define el inicio y fin de una columna o fila.

Ejemplo: grid-column: 1 / 3;

Ejemplo: grid-row: 2 / 4;

grid-area Asigna un área a un elemento.

Ejemplo: grid-area: header;

  • display: grid; Define un contenedor de cuadrícula.
  • grid-template-columns/rows: Define las filas y columnas de la cuadrícula.
  • grid-gap: Define el espacio entre las filas y columnas.
  • grid-area: Asigna un área de la cuadrícula a un elemento.

¿Qué es Bootstrap 5?

Bootstrap 5 es una versión actualizada del popular framework de front-end, que facilita la creación de sitios web responsive y móviles.

Bootstrap es un framework CSS y Javascript diseñado para la creación de interfaces limpias y con un diseño responsive. Además, ofrece un amplio abanico de herramientas y funciones, de manera que los usuarios pueden crear prácticamente cualquier tipo de sitio web haciendo uso de los mismos.

Bootstrap 5, la última versión y más versátil Desde que Bootstrap lanzó su versión 5, se ha convertido en la mejor opción para muchos desarrolladores y aunque se liberó en 2021, aun sigue en plena forma. La última actualización es la 5.2.3 y fue lanzada en noviembre de 2022. por lo que el desarrollo del framework sigue en marcha. Algunas de las mejoras que trajo la versión 5 de Bootstrap son las siguientes: Adiós a la dependencia de jQuery jQuery es una librería de JavaScript. Bootstrap no la utiliza y con esto se ha mejorado el tamaño de los archivos y el rendimiento. Se sigue utilizando JavaScript, obviamente, pero sólo de forma nativa. Personalización mejorada La personalización es imprescindible para cualquier desarrollador. Con Bootstrap 5 puede usar propiedades CSS totalmente personalizadas. Fuentes, colores, dimensiones o cualquier otro tipo de estilo, pudiendo aplicar los cambios en todo el sitio. Mejoras en la rejilla Ya hemos visto que Bootstrap se basa en un sistema de rejilla. En la versión 5 todo ha mejorado, proporcionando muchas más opciones para ajustar las rejillas de cualquier forma y lograr el diseño deseado de la web. Documentación ampliada Imprescindible para cualquier desarrollador, ya que nadie nace sabiendo de todo. La documentación es necesaria para que una plataforma pueda avanzar y continuar con el proceso. En Bootstrap 5 la documentación está mucho más cuidad, lo que anima a los desarrolladores a seguir utilizando el framework. Formularios con más opciones Los formularios son también una parte muy importante de cualquier web. Ahora se disponen de muchas más opciones con las que lograr que el formulario se adapte al diseño, sin faltar las medidas de seguridad, como las casillas de verificación. Biblioteca de iconos Con los nuevos iconos que encontrarás en Bootstrap 5, no necesitarás buscar más en otra fuente, ya que trae más de 1300 iconos que puedes usar totalmente gratis. Hay muchas más mejoras en la versión 5 de Bootstrap, aunque estas suelen ser algunas de las más destacadas.

  • Rejilla (Grid system): Sistema de cuadrícula para el diseño responsivo.
  • Componentes: Amplia colección de componentes pre-estilizados.
  • Utilidades: Clases de utilidad para el espaciado, tipografía y más.
  • JavaScript: Funcionalidades adicionales con plugins de JavaScript.
